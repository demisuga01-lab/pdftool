# Roadmap

## Purpose

Track future work without confusing planned items with implemented ones.

## Audience

- Product owners
- Developers
- Future maintainers

## Scope

Everything in this file is **Planned** unless explicitly moved into implemented documentation elsewhere.

## Planned Items

- Separate API platform at `api.wellfriend.online`
- Auth and API key management for API product
- Paid plans and quota enforcement
- Usage analytics
- Admin dashboard
- Better queue monitoring
- Better usage metering
- More file formats and format combinations
- More refined mobile editor interactions
- Better automated tests
- Better mail reputation and deliverability monitoring
- More observability and alerting
- Optional user accounts
- Team and business features
- Webhook and API documentation for future platform
- More hardening around external file scanning
- Virus scanning if adopted
- User-triggered deletion controls
- Better accessibility coverage
- Internationalization if the product expands language support

## Implementation Status

None of the roadmap items above should be treated as live product capabilities today unless they are also described as implemented elsewhere in this documentation set.
