# Contact and Support

## When to use support@wellfriend.online

Use `support@wellfriend.online` for:

- bugs
- processing failures
- upload or download problems
- security reports
- abuse reports

## When to use contact@wellfriend.online

Use `contact@wellfriend.online` for:

- general questions
- API questions
- business and partnerships
- feature suggestions

## What to include in a report

Include as much of the following as you can:

- tool name
- file type
- approximate file size
- exact error text
- job ID if available
- what you expected to happen

## Community help

For updates and community discussion, you can also join the WellFriend Discord server from the main site.
