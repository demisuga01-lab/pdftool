# PDFTools Help Center

Use this help center to understand what each tool does, how uploads and downloads work, and what to do when something goes wrong.

## Start here

- Read [Getting Started](./getting-started.md) if you are new to PDFTools.
- Open [PDF Tools](./pdf-tools-help.md) to compare common PDF workflows.
- Open [Image Tools](./image-tools-help.md) to compare image workflows.
- Read [File Handling and Privacy](./file-handling-and-privacy.md) before uploading sensitive material.
- Use [Troubleshooting](./troubleshooting-user.md) if a job fails or a result looks wrong.
- Use [Contact and Support](./contact-and-support.md) when you need help from the team.

## File handling

Files are processed temporarily for the selected tool and are not kept as permanent storage.

## Need technical references?

Architecture, backend, deployment, and operational references are still available in the developer and operator section of `/docs`.
