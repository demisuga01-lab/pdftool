# Image Tools

## Resize images

Change image dimensions for web uploads, documents, or social formats.

## Compress images

Reduce image file size while balancing quality and download weight.

## Convert images

Switch between common image formats such as JPG, PNG, WebP, AVIF, TIFF, BMP, or PDF where supported.

## Crop images

Trim away unwanted edges or focus on a specific area.

## Rotate images

Fix sideways images or apply a specific rotation before download.

## Remove background

Create a cutout result for product shots, profile images, or simple graphics. Very fine edges may still need review.

## Watermark images

Apply branding or ownership marks before sharing or publishing.

## Image info

Inspect image dimensions, format, file size, and related details before you process or publish the file.

## Extract text from images

Use OCR on screenshots, scans, and photos that contain readable text. Clean, high-contrast images usually produce better results.
