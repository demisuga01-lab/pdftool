# Mail and Contact

## Purpose

Document mailbox usage and public contact ownership.

## Public Endpoints

- Website: `https://tools.wellfriend.online`
- Mail host: `https://postal.wellfriend.online`

## Mailboxes

- `support@wellfriend.online`: bugs, failures, upload/download issues, abuse, security
- `contact@wellfriend.online`: general contact, API, features, business
- `https://discord.gg/ZQFmYaQbVu`: community help, updates, and discussion

## Channel Rules

- Keep `support@wellfriend.online` as the official path for bugs, processing failures, abuse, and security reports
- Keep `contact@wellfriend.online` for API, product, feature, and business conversations
- Use Discord for community discussion and quick help, not as a guaranteed official support SLA

## Mail/DNS Notes From Current Project Context

- aliases may forward operational addresses to support
- public production docs should not include admin passwords
- Cloudflare DNS-only handling may matter for mail hosts
