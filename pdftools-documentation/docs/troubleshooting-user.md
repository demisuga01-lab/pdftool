# Troubleshooting

## Upload failed

- Check your internet connection and retry.
- Confirm that the file matches the tool you selected.
- Retry with a different browser tab if the first upload appears stuck.

## File too large

- Try a smaller file.
- Split a large PDF into parts before processing.
- Compress the original image or PDF before using another tool.

## Processing failed

- Retry once to rule out a temporary problem.
- Check whether the file is password-protected, damaged, or unusually complex.
- Try a simpler output option if multiple formats are available.

## Download unavailable

- Wait a few moments and try the download again.
- Check whether the job is still processing.
- If the result panel shows an error, report it with the job ID.

## Rate limit message

The public website limits how many requests can be made in a period of time. Wait for the limit to reset, then retry.

## OCR did not detect text

- Use a cleaner scan or sharper image.
- Increase contrast before upload if the source is faint.
- Confirm that the text is large enough to read visually.

## Watermark is not visible

- Increase watermark size or opacity.
- Move it away from very bright or very dark areas.
- Check whether the watermark is behind other content in the preview.

## Converted file does not look right

- Complex layouts, fonts, tables, and embedded graphics may not convert perfectly.
- Try a different output format if available.
- Review the source file for unusual formatting before retrying.

## Browser preview does not load

- Refresh the page.
- Retry with a smaller file.
- Try a different browser if the preview keeps failing while the upload succeeded.

## When to contact support

If the same file fails repeatedly, contact support and include the tool name, file type, approximate size, exact error text, and job ID if it is shown.
